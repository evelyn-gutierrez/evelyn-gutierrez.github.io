Baby's 3D model using SfM.

Last updated on Jul 28, 2022 2 min read

A baby’s 3D model using Structure From Motion (SfM).

Following a photo session of a baby, I made the 3D reconstruction to create a nice gift for the mother.

The input was a 14 seconds video from which 32 frames (480x848px) were extracted to create the 3D model with Structure From Motion.

I made the reconstruction using two free 3d reconstruction software:

  • Meshroom
  • 3DF Zephyr (free version)

I usually use Meshroom but I read that you also get good results with 3DFlow Zephyr, so I did the reconstruction and chose the better one.

  • Meshroom took betwen 25 to 30 minutes with default settings and high sensitivity for feature extraction.

  • 3DFlow Zephyr took about 5 to 7 minutes. I used the default settings, higher and ultra sensitivity for feature extraction and dense reconstruction.

The point cloud density was higher with 3DFlow Zephyr (~20K points 3DFlow Zephyr vs ~16K points with Meshroom), however, I found qualitatively better results with Meshroom: The baby’s forehead appears sunken in the 3DFlow Zephyr reconstruction. In contrast, Meshroom showed more solidity in the reconstruction in this case, and the forehead did not become sunken. This problem was due to lighting changes in this area during video capture.

Finally I removed some background object manually to have the final 3D model and added a Laplacian smoothing.

The result was uploaded to SketchFab which allows quick visualization and interaction. Here is the result:

Baby Gift by egutierreza on Sketchfab

This model was later 3D printed and made a nice gift for the mother.
