Introducción a Computer Vision

Última actualización el Aug 21, 2021 2 min de lectura computer vision

Enfoques:

Hay dos formas de realizar el procesamiento de imágenes:

  1. De forma tradicional.

De forma tradicional, obtenemos caracteristicas de la imágen mediante processos manuales, y luego entrenamos modelos simples de clasificación.

  1. Con Deep Learning (DL).

Bajo este enfoque, las features y modelos de clasificación son entrenados y requiren gran cantidad de imágenes. Los modelos a entrenar son de mayor complejidad y por ello requieren más imágenes que el enfoque tradicional.

Problemas que podemos resolver:

  1. Detección y clasificación:

Por ejemplo: detección de un objeto, de una clase de objeto, o de contornos. En este caso, es enfoque más utilizado es usando Deep Learning.

  1. Análisis de movimiento:

Optical flow y object tracking se encuentran en esta categoría.

  1. Reconstrucción de escena. (Scene reconstruction)

Reconstruir el la forma y aspecto en 3D de un objeto a partir de imágenes. También se llama 3D reconstruction.

  1. Restauración:

Aquí se realizan procedimientos para mejorar las imágenes como estos:

  • Reducir el ruido de las imágenes,
  • Debluring, o
  • Aumentar la resolución de imágenes.

Librerías (Python)

Librerías que podemos utilizar para comenzar:

  • En general:

    • OpenCV
    • Pillow, PIL

  • Para 3D:

    • PCL
    • Open3D

  • Para DL. Frameworks:

    • TensorFLow
    • Pytorch
    • Caffe (y más)

Aprender más:

Para aprender más sobre computer visión, se puede leer los siguientes libros:

  • Simon J.D. Prince, Computer vision: models, learning and inference

  • Richard Szeliski, Computer vision: algorithms and applications
